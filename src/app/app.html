Flexible & Parametrized Animations

Each animation is highly customizable with parameters. Instead of having separate animations like fadeInLeft, fadeInRight, fadeInUp, etc., we have one powerful animation that you can customize:

fadeIn // Simple fade
fadeIn({{ '{' }} x: '-100%' {{ '}' }}) // Fade from left
fadeIn({{ '{' }} x: '100%' {{ '}' }}) // Fade from right
fadeIn({{ '{' }} y: '100%' {{ '}' }}) // Fade from bottom
fadeIn({{ '{' }} x: '-100%', y: '-100%' {{ '}' }}) // Fade from top-left corner

✨ Check the examples below each animation to see how to customize them!
