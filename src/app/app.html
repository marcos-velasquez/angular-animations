// In your component
@Component({{ '{' }}

Available Directives

animateLoad

Triggers animation when the element loads.

<div animateLoad="fadeIn"></div>

animateClick

Triggers animation on click.

<button animateClick="pulse">Click me</button>

animateEnter

Triggers animation on mouse enter.

<div animateEnter="shake">Hover me</div>

animateLeave

Triggers animation on mouse leave.

<div animateLeave="fadeOut">Hover me</div>

Animation Syntax

Animations can be customized with parameters using object notation:

// Simple animation
<div animateClick="fadeIn"></div>
// With parameters
<div animateClick="fadeIn({{ '{' }} x: '-100%', opacity: 0 {{ '}' }})"></div>
// Multiple animations (sequence)
<div animateClick="fadeIn | scale: 1.2 | rotate: 360"></div>

Common Parameters

Position

  • x - Horizontal position
  • y - Vertical position

Scale

  • scale - Uniform scale
  • scaleX - Horizontal scale
  • scaleY - Vertical scale

Rotation

  • rotate - 2D rotation
  • rotateX - 3D X-axis rotation
  • rotateY - 3D Y-axis rotation

Opacity

  • opacity - Transparency (0-1)

Timing

  • duration - Animation duration
  • delay - Start delay
  • ease - Easing function

Skew

  • skewX - Horizontal skew
  • skewY - Vertical skew

Examples

Fade in from left

<div animateLoad="fadeIn({{ '{' }} x: '-100%' {{ '}' }})">Content</div>

Bounce on click

<button animateClick="bounceIn">Click me</button>

Rotate and scale on hover

<div animateEnter="zoomInRotate({{ '{' }} scale: 1.2, degrees: 180 {{ '}' }})">Hover me</div>

Complex sequence

<div animateClick="fadeIn | scale: 1.2 | rotate: 360 | fadeOut">Complex</div>
